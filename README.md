To configure your current shell, you need to source
the corresponding env file under $HOME/.cargo.
<br>
This is usually done by running one of the following (note the leading DOT):
. "$HOME/.cargo/env"            # For sh/bash/zsh/ash/dash/pdksh
source "$HOME/.cargo/env.fish"  # For fish
source "~/.cargo/env.nu"        # For nushell
source "$HOME/.cargo/env.tcsh"  # For tcsh
. "$HOME/.cargo/env.ps1"        # For pwsh
source "$HOME/.cargo/env.xsh"   # For xonsh

#### Fish

In the context of the Rust programming language, "fish" almost always refers to the Turbofish operator, written as ::<>, 
which is used to explicitly provide concrete types to generic functions or methods. Alternatively, it could refer to the popular command-line tool fish shell, 
which was recently completely rewritten in Rust.
<br>
The Turbofish Operator (::<>)Rust relies heavily on type inference to figure out types automatically. 
However, when the compiler does not have enough context to guess a generic type, you use the Turbofish Operator to state it explicitly

Nushell (or "Nu") is a modern, cross-platform command-line shell that treats all output as structured data (tables, records, and lists) rather than plain text
